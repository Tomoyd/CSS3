# CSS3
> 边框
- 主要涉及边框的圆角边框，阴影和图片
    -  border-radius
    -  box-shadow
    -  border-image
        * border-image-source 图片资源
        * border-image-slice 图片向内偏移
        * border-image-width 图像边界宽度
        * border-image-outset 指定边框外部绘制border-image-area的量
        * border-image-repeat :可选值：repeat，stretch 铺满(round)
